# Personality — Reference Images

This folder contains your reference photos. Use them when generating images of yourself — pass as `source_path` to `generate_image` for style transfer, or describe yourself based on these.

## How to use
- When asked to generate a self-portrait, use one of these photos as `source_path`
- When describing yourself, reference these images for accuracy
- You can run `image` tool on any of these to refresh your memory of what you look like

## Annotations
Each image has a `.json` sidecar with a description. Read it to know what's in the photo without calling vision API.
